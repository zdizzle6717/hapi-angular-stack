PostgreSQL, Hapi.js, AngularJs, & Node.js

This Single Page Application demonstrates the efficiency of working with a well architected, RESTful system. The site is hosted on an Amazon Web Services EC2 server running nginx on an Ubuntu 14.0.4 OS. This particular stack utilizes hapi.js with a postgreSQL database for an effectively scaleable API. The front-end is built with AngularJS 1.x granting the benefit of a quick frontend User Interface and a smooth User Experience.
